Tools for syntactic and lexical-semantic analysis across theoretical frameworks.

Telperion

An interactive builder for RRG trees, with constituent, operator, and information structure projections of the Layered Structure of the Clause.

Syntax / RRG

Ent

An interactive builder for constituency trees, with a built-in catalog of function and category labels and a practice mode with model comparison.

Constituency / Pedagogy

Kilmer

An interactive builder for Generative Grammar trees, with movement arrows, phase boundaries, and an IPA and special characters palette.

Generative Syntax

MAK

An interactive builder for SFL trees, with support for multi-layered functional analysis and clause-level semantic structure.

Syntax / SFL

Albura

A declarative generator for RRG Layered Structure of the Clause diagrams, with constituent and operator projections at the simple clauses level.

Syntax / RRG

Vendler

An interactive wizard for Aktionsart classification within RRG, extending the Vendlerian typology, with a 5-feature decision matrix and logical structure generation.

Lexical Semantics
